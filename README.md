## Getting started

### Run packages
```
npm install
```

### Copy RDF files

Put contents (`cache` folder) of `rdf-files.zip` into the project folder.

### Migrate DB

- add DB details to `config/config.json` and `.env`
- run `npm run migrate` to add `Meta` table, which will store all the book meta data

### Run the script

```
npm start
```

### Run tests

```
npm test
```
